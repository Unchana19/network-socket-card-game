## Card Game - Pockdeng
pockdeng game for 2 players.



## Features

- draw 2 cards
- draw addition card
- compare 2 players cards


## Usage/Examples
#### Server

```javascript
cd server
```

```javascript
python app.py
```

#### Client *(run 2 clients to play)*

```javascript
cd client
```

```javascript
python app.py
```
